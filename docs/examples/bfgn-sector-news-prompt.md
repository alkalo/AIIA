# Example — Sector news aggregator (generic / BFGN-ready)

Use **Gemini + high/ultra** or local Máx. Destinations: **Inbox**.

```
Aggregate sector news for purpose-led / business-for-good changemakers (last ~30 days).

Topics: social enterprise, B Corp, philanthropy, NGO sector, impact investing,
ESG/CSR with sector relevance, First Nations economic development, major funding pools,
mergers/closures, awards, partnerships.

Rules:
- Prefer Australian national coverage; include state/global only if sector-relevant.
- Each item: title, summary, source, publication_date, url (https), why_it_may_matter, score.
- Australian English. Do not invent URLs or dates.
- Do NOT include open grant listings (use a separate Opportunities agent for those).
- Aim for ~10–20 high-quality candidates; quality before quantity.

Review in Inbox (Pending → Approve / Reject). Newsletter formatting is optional later.
```

Set opportunity subtype to **Sector news**.
